## Install & Usage
### Install
`git clone https://github.com/ildison/Lem-in.git Lem-in`  
`cd Lem-in`
### Usage

`./lem-in [flag] < [map]`

Flags:

`-d - for debug information`

`-v - for visualisation`

Example: `./lem-in -v < tests/sferical_test_in_vacuum`

### Control keys for visualisation mode:
`Play/pause     - space` 

`prev/next step - "pause" left/right`

`speed          - up/down`
